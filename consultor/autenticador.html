Administrador

Como administrador usted puede crear, modificar y eliminar usuarios, roles, servicios y servicios de autenticación. Si lo desea, también puede asignar y eliminar roles a los usuarios.

Users

Esta entidad representa a los posibles usuarios globales en la institución.

Rols

Esta entidad representa los tipos de roles o cargos que un usuario puede tener o desempeñar en la organización.

Services

Esta entidad representa un servicio o sistema que utiliza al servidor de autenticación. De esta manera, un usuario puede estar ligado a un servicio de autenticación para tener una contraseña específica para dicho servicio.

Userservice

Esta entidad liga usuarios y servicios de autenticación. Esta entidad le permite a un usuario tener formas individuales para autenticarse en diferentes servicios.
